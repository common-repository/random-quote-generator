== Description ==
A plugin to generate random quotes to be shown to the user at any page via a shortcode.

== Installation ==

1. Upload the plugin in the `/wp-content/plugins/` directory, or automatically install it through the 'New Plugin' menu in WordPress
2. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= How can I use this plugin? =

This plugin provides a shortcode "[quote]" wich you can add in the content of any page of your website.
